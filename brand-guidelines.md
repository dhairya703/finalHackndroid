# HacknDroid Brand Guidelines

## 1. Typography

The HacknDroid brand relies on sleek, modern typography that balances legibility with a futuristic tech vibe.

*   **Primary Font:** `Outfit` (sans-serif)
    *   **Weights:** Regular (400), Bold (700), Black (900)
    *   **Usage:** Body text, taglines, subheadings, and general UI elements.
*   **Display/Accent Font:** `Bebas Neue`
    *   **Usage:** Hero logs, prominent display text, versioning indicators, and high-impact visual sections.

## 2. Color Palette

Our color palette is anchored in a dark mode aesthetic with vibrant, neon-like accents to communicate innovation, energy, and a cyberpunk/tech feel.

### Core Colors
*   **Background (Canvas):** `#000000` (Pitch Black)
*   **Primary Text:** `#ffffff` (Pure White)
*   **Muted Text:** `rgba(255, 255, 255, 0.6)`

### Accent Colors (Neon Palette)
*   **Primary Brand:** `#a855f7` (Neon Purple)
*   **Secondary Brand:** `#ec4899` (Hot Pink)
*   **Accent Brand:** `#22d3ee` (Cyan / Light Blue)

### UI Surfaces & Borders
*   **Surface Backgrounds:** `rgba(255, 255, 255, 0.05)` — Used for cards, panels, and subtle depth on the black background.
*   **Card/Container Borders:** `rgba(255, 255, 255, 0.1)` — Used to give definition to floating elements without overpowering the clean look.

## 3. Visual Styling & Effects

### Glows and Drop Shadows
A core part of the HacknDroid aesthetic is the use of neon glows (`rgba` variations of accent colors).
*   **Primary Glow:** `rgba(168, 85, 247, 0.4)`
*   **Secondary Glow:** `rgba(236, 72, 153, 0.4)`
*   **Accent Glow:** `rgba(34, 211, 238, 0.4)`

### Gradients
*   **Logo / Hero Text Gradient (Droid):** `#e9d5ff` stop 0%, `#a855f7` stop 30%, `#7e22ce` stop 50% to `#f3e8ff`.
*   **Monochrome Text Gradient (Hack):** Smooth silver gradients (`#f8f9fa` to `#adb5bd`) with bright white highlights.

## 4. UI Layout Characteristics
*   **Border Radius:** Rounded pill shapes (`9999px`) for buttons/navbars and standard rounded corners for cards.
*   **Layering:** Extensively uses backdrop filters (e.g., `blur(16px)`, `blur(24px)`) to create a glassmorphism effect.
*   **Animations:** Focus on smooth, deliberate micro-actions such as subtle pulse elements and orbital spins to reflect a constant "active" state.
